# Mini Agile Workflow App

A lightweight, local-first Kanban project management tool built with React and @dnd-kit.

## Features
- **Project Management**: Create and manage multiple projects.
- **Kanban Board**: Drag and drop tasks between columns (Backlog, To Do, In Progress, Done).
- **Task Management**: Detailed task creation with priorities, due dates, and assignees.
- **Persistence**: Data is automatically saved to LocalStorage.

## Setup

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- @dnd-kit (Drag and Drop)
- Lucide React (Icons)
